Notes de test
Xamarin.iOS
  • Le moyen le plus simple pour démarrer est d’utiliser les simulateurs iPhone, iPad, Apple Watch et Apple TV inclus dans Xcode. Les simulateurs sont accessibles sur le Mac connecté pendant le débogage avec Visual Studio.
  • Pour utiliser un appareil de test, suivez ces instructions (en suivant plus d’instructions pas à pas sur l’ordinateur Mac connecté).
Xamarin.Android Suivez ces instructions pour configurer votre appareil ou utilisez un émulateur :
  • Le programme d’installation Xamarin inclut Google Emulator Manager qui vous permet de configurer les émulateurs Google Android pour les tests.
  • D’autres émulateurs sont disponibles à partir de Genymotion et Microsoft (avec Visual Studio 2015).
Xamarin.Forms Les applications Xamarin.Forms peuvent être déployées sur les appareils et émulateurs appropriés, comme décrit ci-dessus. L’application iOS peut uniquement être testée via le matériel Mac connecté ; et les applications de tablette/bureau Windows pour Windows 8.1 ou Plateforme Windows universelle peuvent être testées directement sur l’ordinateur de développement.
